# 🐰 Tug-of-WAR-X 🐻

A real-time multiplayer math game built with Firebase.

## How to Play
1. Open the game link
2. Enter your name and click **Create Room**
3. Share the 6-character room code with a friend
4. They enter the code and join
5. Answer multiplication questions as fast as you can!

## Rules
- ✅ Correct answer = +1 unit (speed bonus up to +2)
- ❌ Wrong answer = -2 units
- ⏰ No answer in 5 seconds = -2 units
- First to pull the rope past the threshold wins!

## Built With
- Vanilla HTML/CSS/JavaScript
- Firebase Realtime Database
